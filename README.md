# Django Blog App

A full-featured blog application built with Django, PostgreSQL, Tailwind CSS, and Django Authentication.

## Live Demo

Live Project: https://your-render-url.onrender.com/

## Features

- User Registration
- User Login / Logout
- Create, Read, Update, Delete Blog Posts
- Category-based Filtering
- Search Functionality
- Pagination
- Comment System
- Django Messages Framework
- PostgreSQL Database
- Welcome Email using Django Signals
- Admin Panel
- Responsive UI with Tailwind CSS
- Deployed on Render

## Tech Stack

- Python
- Django
- PostgreSQL
- HTML
- Tailwind CSS
- Gunicorn
- WhiteNoise
- Render
